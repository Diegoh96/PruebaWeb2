# PruebaWeb2
# Actividad 2.6 - Backend + Frontend + Cheerio + MySQL

## Descripción

Proyecto desarrollado en Node.js utilizando Express y Cheerio para procesar HTML en backend, extraer información de productos y mostrarla en un frontend conectado mediante fetch API.

El sistema utiliza arquitectura por capas, conexión a base de datos MySQL, manejo de errores y validaciones básicas.

---

# Tecnologías Utilizadas

- Node.js
- Express
- Cheerio
- MySQL
- HTML5
- CSS3
- JavaScript
- Fetch API

---

# Objetivo

Procesar HTML utilizando Cheerio en backend para extraer información estructurada y mostrarla en un frontend conectado a una base de datos.
